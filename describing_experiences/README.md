Describing Experiences - Interactive ESL Lesson

This repository contains an interactive, web-based adaptation of an ESL lesson focused on "Describing Experiences." Designed strictly for enrolled Low Intermediate English language learners (ESL 404/412), this single-page application transforms standard paper/PDF curriculum into an engaging, digital experience with instant feedback.

Legal & Fair Use Notice: This project is intended solely for educational use by students currently enrolled in the instructor's courses under an active institutional ELLII license. To comply with copyright and fair use guidelines, access to the lesson's content is restricted via a class-specific access code. It is not intended for open public distribution, commercial use, or external reuse.

🌟 Key Features

Interactive Brainstorming: A dynamic input section where students can type out, display, and delete their own ideas directly on the page.

Embedded Audio Dialogues: Seamlessly integrated audio players (.mp3) for listening to dialogues about great, disappointing, and interesting experiences.

Instant Grammar Feedback: A self-checking dropdown activity for practicing -ed vs -ing participial adjectives. The fields instantly turn green for correct answers and red for incorrect ones.

Self-Assessing Listening Comprehension: A multiple-choice listening quiz that provides immediate visual feedback to help students self-assess their understanding.

Custom "Terracotta" Theme: Designed with a warm, accessible color palette that is easy on the eyes and provides clear contrast for readability.

Access Code Protection: Features a simple entry overlay to keep students focused and ensure they are in the right place.

🚀 How to Use

Hosting: This project is built using standard HTML, CSS, and vanilla JavaScript. It can be easily hosted using GitHub Pages.

Audio Files: Ensure that the four associated .mp3 audio files are located in the exact same directory as the describing_experiences.html file so the relative links work correctly.

Access Code: When prompted at the start of the lesson, students must enter the class code: 412 to access the materials.

📁 File Structure

describing_experiences.html - The main interactive lesson page.

Describing-Experiences-D1.mp3 - Audio for Dialogue 1.

Describing-Experiences-D2.mp3 - Audio for Dialogue 2.

Describing-Experiences-D3.mp3 - Audio for Dialogue 3.

describing-experiences-listening.mp3 - Audio for the listening comprehension quiz.
